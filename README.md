# Rexie Maris Bag Manufacturing - Production Management System

## Project Overview

A new and revised version of the original production management system made for Rexie Maris Bag Manufacturing Company in 2023. This comprehensive web-based dashboard provides complete oversight and control over bag production operations, inventory management, client relations, and business analytics.

## About Rexie Maris Bag Manufacturing

Rexie Maris is a specialized bag manufacturing company that produces high-quality bags including totes, briefcases, backpacks, and clutches. This production management system was designed to streamline their operations and provide real-time insights into their manufacturing processes.
